Aim

The aim of this section is to get you familiar with the Rainbird scenario which we are using today.

By the end you should understand what Rainbird is doing as well as having seen two of the interfaces Rainbird provides.

If you are unsure of how to complete any of the tasks please ask.

Instructions

Step 1

Edit this html file using Visual Studio.

Add an iframe to the code using the following information:

  • src attribute: https://app.rainbird.ai:443/agent/4fb4c8d2-6cdf-41a4-aecb-555bb0d78f60
  • width attribute: 500
  • height attribute: 600

Place the iframe markup in the <section> tag with the class="right" property.

You may find this webpage useful: https://developer.mozilla.org/en-US/docs/Web/HTML/Element/iframe (scroll down to see an example)

Step 2

Refresh the html file in your browser to see a Rainbird interface.

(Please ask if you don't see any changes).

Run through the Rainbird interaction by first selecting the 'Recommend a Coffee Shop' query. Answer any questions Rainbird asks you.

Rainbird will recommend you some cafes you could visit.

Have a look at the Evidence Tree by clicking the 'i' icon. This will show you how Rainbird has made its decision.

Behind the scenes Rainbird has taken the answers you gave to obtain your direction of travel (bearing) and used other information such as Google Star Ratings and Food Hygiene scores to produce a list of recommended cafes.

Step 3

The interface you have used so far to interact with Rainbird is just one option.

To see another style of interface update the iframe you added replace '/agent/' with '/chatagent/' in the url used as the src attribute.

Refresh the page to reload the html and feel free to run the query again if you like.

Finished

Well done! You finished this section.
