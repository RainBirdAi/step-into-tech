Aim

The aim of this section is to get you familiar with the Rainbird scenario which we are using for this project.

By the end you should understand what Rainbird is doing as well as having seen two of the interfaces Rainbird provides.

If you are unsure of how to complete any of the tasks please ask us or a friend.

Instructions

Step 1

Edit this html file using Visual Studio.

Add an iframe to the code using the following information:

  • src attribute: https://app.rainbird.ai/agent/7fa42734-3586-4777-9779-bfbba84141ab
  • width attribute: 500
  • height attribute: 600

Place the iframe markup in the <section> tag with the class="right" property.

You may find this webpage useful: https://developer.mozilla.org/en-US/docs/Web/HTML/Element/iframe (scroll down to see an example)

Step 2

Refresh the html file in your browser to see a Rainbird interface.

(Please ask if you don't see any changes).

Run through the Rainbird interaction by first selecting the 'BLAH BLAH' query. Answer any questions Rainbird asks you.

Rainbird will recommend you some cafes you could visit.

Copy and Paste the cafe name here if you like, or just remember it:

Behind the scenes Rainbird has taken the answers you gave to obtain your direction of travel (bearing) and used other information such as Google Star Ratings and Food Hygine scores to produce a list of recommended cafes.

Step 3

Run the Rainbird interaction by selecting the 'BLAH BLAH' query.

Rainbird will generate you a Google Maps link showing how to get to your chosen cafe.

Once the interaction is complete, open a new browser tab and copy and paste in the result URL.

Step 4

Did you notice that we had to tell Rainbird the two pieces of location information when we ran our second query?

In fact Rainbird has the ability to remember certain pieces of information, but we have to tell it that this is how we'd like it to behave.

Change the src attribute of the iframe you added and add a query string to the end of the url.

The query string should have the parameter name contextid and the value should be something unique to you e.g. your name followed by your favorite colour and favorite number.

e.g. contextid=craig-orange-13

You may find this webpage useful: https://mixpanel.com/blog/2014/10/16/community-tip-adding-properties-into-the-url/ (see What is a query string?)

Step 5

Refresh the page and run the 'BLAH BLAH query again.'

Choose a cafe as you did before, remmeber it or paste it here:

Now run the 'BLAH BLAH' query.

This time you shouldn't have had to tell Rainbird your location information. It has remebered those details.

Step 6

The interface you have used so far to interact with Rainbird is just one option.

To see another sytle of interface update the iframe you added replace '/agent/' with '/chatagent/' in the url used as the src attribute.

Refresh the page to reload the html and feel free to run the queries again if you like.

Finished
